# Nuxt MCP Server 活用ガイド

## 利用可能なMCPツール

| ツール | 用途 |
|--------|------|
| `list_documentation_pages` | ドキュメント一覧取得（カテゴリ・バージョンでフィルタ可） |
| `get_documentation_page` | 特定ページの内容取得（パス指定） |
| `get_getting_started_guide` | バージョン別の入門ガイド取得 |
| `list_blog_posts` | ブログ記事一覧取得 |
| `get_blog_post` | 特定ブログ記事の内容取得 |
| `list_deploy_providers` | デプロイプロバイダ一覧取得 |
| `get_deploy_provider` | 特定プロバイダのデプロイ手順取得 |

## ワークフロー

### 1. 開発中のドキュメント参照

Nuxtの機能・API・設定について調べる時：

1. `list_documentation_pages` でカテゴリを絞り込み、該当ページを特定する
   - `version` パラメータでバージョンを指定する（デフォルト: 4.x）
2. `get_documentation_page` でページ内容を取得する

**ポイント**: 漠然と検索せず、まず一覧でカテゴリを絞ってから詳細ページを取得する。

### 2. デプロイ設定

Nuxtアプリのデプロイ先を選定・設定する時：

1. `list_deploy_providers` で対応プロバイダ一覧を取得する
2. ユーザーの要件（ホスティング環境、予算、スケーラビリティ等）に合うプロバイダを提案する
3. `get_deploy_provider` で選定プロバイダの具体的なデプロイ手順を取得する
4. 手順に従ってプロジェクトのデプロイ設定を行う

### 3. バージョンマイグレーション

Nuxtのバージョンを移行する時（例: 3.x → 4.x）：

1. `get_getting_started_guide` で移行先バージョンのガイドを取得する
2. `list_documentation_pages` で移行先バージョンのドキュメント一覧を取得し、Breaking Changesやマイグレーションガイドを探す
3. `get_documentation_page` で関連ドキュメントの詳細を取得する
4. 現在のプロジェクトコードと照合し、必要な変更箇所を特定・実施する

### 4. 最新情報の確認

Nuxtの最新リリースやアップデート情報を確認する時：

1. `list_blog_posts` でブログ記事一覧を取得する
2. リリースノートやアップデート関連の記事を特定する
3. `get_blog_post` で詳細を取得する
4. プロジェクトへの影響を評価し、必要な対応を提案する
